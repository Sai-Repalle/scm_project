# CVER
self managed configuration mangement(scm) is a simple commandline app for configuration management
# Installation
## Using Pip
```bash
  $ pip install scm
```
## Manual
```bash
  $ git clone https://github.com/citguru/cevr
  $ cd cver
  $ python setup.py install
```
# Usage
```bash
$ cver
```
## Search
`search <keyword>`
```bash
$ cver search python
```
## Lookup
`search <name>`
```bash
$ cver look-up CVE-2020-2121
```
